# Guess-The-Word-Game

A simple Wordle-style game built with HTML, CSS, and JavaScript.
The player has 6 attempts to guess the hidden word, with color-coded feedback for each letter and an optional hint.
This project demonstrates DOM manipulation, game logic, and an interactive UI.
